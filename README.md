# 다음 분기에 어떤 게임을 설계해야 할까
data url : https://ds-lecture-data.s3.ap-northeast-2.amazonaws.com/datasets/vgames2.csv

Data Description

Name : 게임의 이름입니다.
Platform : 게임이 지원되는 플랫폼의 이름입니다.
Year : 게임이 출시된 연도입니다.
Genre : 게임의 장르입니다.
Publisher : 게임을 제작한 회사입니다.
NA_Sales : 북미 매출액입니다.
EU_Sales : 유럽 매출액입니다.
JP_Sales : 일본 매출액입니다.
Other_Sales : 기타지역 매출액입니다.
